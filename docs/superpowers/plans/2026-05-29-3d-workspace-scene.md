# 3D Workspace Scene Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a persistent, scroll-driven 3D workspace scene (laptop + monitor + lamp + plant) as the portfolio's new background, replacing the current 2D FX layers, with cinematic camera choreography per section and tiered perf for mobile.

**Architecture:** Single full-viewport `<Canvas>` mounted at the app root behind all content. Lenis is lifted into a provider that broadcasts scroll progress. `ScrollCamera` interpolates between per-section camera/lighting keyframes anchored to DOM via `IntersectionObserver`. `frameloop="demand"` keeps the GPU idle between scrolls. The entire 3D layer is a lazy chunk; first paint cost is zero. A single `usePerfTier()` hook governs quality knobs across all tiers (low/mid/high).

**Tech Stack:** React 18, Vite 4, Tailwind 3, existing Framer Motion 10 + Lenis. New: `three`, `@react-three/fiber`, `@react-three/drei`, `@react-three/postprocessing`, `framer-motion-3d`, `@react-three/eslint-plugin`. Model pipeline: `gltf-transform` CLI + `gltfjsx` CLI.

**Testing strategy:** No automated 3D tests (per spec §10). Verification is **manual visual + perf checks at `http://localhost:5173`** at each task's checkpoint. The dev server stays running across tasks; Vite HMR reflects changes immediately. Each task's "Verify" step lists the exact thing to look at. Phase-end ship gates verify Lighthouse + GPU idle behaviour against the spec's success criteria.

**Spec reference:** `docs/superpowers/specs/2026-05-29-3d-workspace-scene-design.md`

Create `src/lib/lenisContext.jsx`:

```jsx
import {
  createContext,
  useContext,
  useEffect,
  useRef,
  useState,
  useSyncExternalStore,
} from 'react';
import Lenis from 'lenis';

const LenisContext = createContext(null);

export function LenisProvider({ children }) {
  const lenisRef = useRef(null);
  const progressRef = useRef(0);
  const listenersRef = useRef(new Set());
  const [enhanced, setEnhanced] = useState(false);
  const [reduced, setReduced] = useState(false);

  useEffect(() => {
    const reducedMq = window.matchMedia('(prefers-reduced-motion: reduce)');
    const enhanceMq = window.matchMedia('(min-width: 1024px) and (hover: hover)');
    const cores = navigator.hardwareConcurrency || 8;

    const recalcEnhanced = () =>
      setEnhanced(!reducedMq.matches && enhanceMq.matches && cores >= 4);
    const recalcReduced = () => setReduced(reducedMq.matches);

    recalcReduced();
    recalcEnhanced();

    enhanceMq.addEventListener('change', recalcEnhanced);
    reducedMq.addEventListener('change', recalcReduced);
    reducedMq.addEventListener('change', recalcEnhanced);

    if (reducedMq.matches) {
      return () => {
        enhanceMq.removeEventListener('change', recalcEnhanced);
        reducedMq.removeEventListener('change', recalcReduced);
        reducedMq.removeEventListener('change', recalcEnhanced);
      };
    }

    const lenis = new Lenis({
      duration: 1.6,
      easing: (t) => Math.min(1, 1.001 - Math.pow(2, -10 * t)),
      smoothWheel: true,
      wheelMultiplier: 0.85,
      touchMultiplier: 1.5,
    });
    lenisRef.current = lenis;

    lenis.on('scroll', ({ progress }) => {
      progressRef.current = progress;
      listenersRef.current.forEach((fn) => fn());
    });

    let rafId;
    const raf = (time) => {
      lenis.raf(time);
      rafId = requestAnimationFrame(raf);
    };
    rafId = requestAnimationFrame(raf);

    return () => {
      enhanceMq.removeEventListener('change', recalcEnhanced);
      reducedMq.removeEventListener('change', recalcReduced);
      reducedMq.removeEventListener('change', recalcEnhanced);
      cancelAnimationFrame(rafId);
      lenis.destroy();
      lenisRef.current = null;
    };
  }, []);

  return (
    <LenisContext.Provider
      value={{ lenisRef, progressRef, listenersRef, enhanced, reduced }}
    >
      {children}
    </LenisContext.Provider>
  );
}

export function useLenisRaw() {
  const ctx = useContext(LenisContext);
  if (!ctx) throw new Error('useLenisRaw must be used inside <LenisProvider>');
  return ctx.lenisRef.current;
}

export function useLenisProgress() {
  const ctx = useContext(LenisContext);
  if (!ctx) throw new Error('useLenisProgress must be used inside <LenisProvider>');
  return useSyncExternalStore(
    (cb) => {
      ctx.listenersRef.current.add(cb);
      return () => ctx.listenersRef.current.delete(cb);
    },
    () => ctx.progressRef.current,
    () => 0,
  );
}

export function useEnhanced() {
  const ctx = useContext(LenisContext);
  if (!ctx) throw new Error('useEnhanced must be used inside <LenisProvider>');
  return ctx.enhanced;
}

export function useReducedMotionPref() {
  const ctx = useContext(LenisContext);
  if (!ctx) throw new Error('useReducedMotionPref must be used inside <LenisProvider>');
  return ctx.reduced;
}
```

**Note on `useSyncExternalStore`:** This is the React 18 API for subscribing to external sources. We expose progress via a ref + listener set so React only re-renders consumers when progress actually changes (every Lenis frame), not every render of the provider.

```js
import { useEffect, useState } from 'react';

export function usePerfTier() {
  const [tier, setTier] = useState('high');

  useEffect(() => {
    const cores = navigator.hardwareConcurrency ?? 8;
    const deviceMemory = navigator.deviceMemory ?? 8;
    const isCoarsePointer = window.matchMedia('(pointer: coarse)').matches;
    const isNarrow = window.matchMedia('(max-width: 1024px)').matches;

    let next = 'high';
    if (cores < 4 || deviceMemory < 4) next = 'low';
    else if (isCoarsePointer || isNarrow) next = 'mid';

    setTier(next);
  }, []);

  return tier;
}

export const TIER_CONFIG = {
  low: {
    dprMax: 1,
    shadows: false,
    shadowMapSize: 0,
    postProcessing: false,
    antialias: 'fxaa',
    environmentResolution: 256,
    showPlant: false,
    showLamp: false,
    contactShadows: false,
    idleJitter: false,
  },
  mid: {
    dprMax: 1.5,
    shadows: true,
    shadowMapSize: 512,
    postProcessing: false,
    antialias: 'msaa2',
    environmentResolution: 512,
    showPlant: true,
    showLamp: true,
    contactShadows: true,
    idleJitter: true,
  },
  high: {
    dprMax: 2,
    shadows: true,
    shadowMapSize: 1024,
    postProcessing: true,
    antialias: 'msaa4',
    environmentResolution: 1024,
    showPlant: true,
    showLamp: true,
    contactShadows: true,
    idleJitter: true,
  },
};
```

`TIER_CONFIG` is the **single source of truth** for the quality-knob matrix (spec §7.2). All scene components read from this exported map.

```jsx
import { Canvas } from '@react-three/fiber';
import { usePerfTier, TIER_CONFIG } from './hooks/usePerfTier';
import { useReducedMotionPref } from '../lib/lenisContext';

export default function SceneCanvas() {
  const tier = usePerfTier();
  const config = TIER_CONFIG[tier];
  const reduced = useReducedMotionPref();

  return (
    <div
      aria-hidden="true"
      className="pointer-events-none fixed inset-0 z-0"
      style={{ background: '#05060A' }}
    >
      <Canvas
        frameloop={reduced ? 'never' : 'demand'}
        dpr={[1, config.dprMax]}
        gl={{
          antialias: config.antialias === 'msaa4' || config.antialias === 'msaa2',
          alpha: false,
          powerPreference: 'high-performance',
        }}
        shadows={config.shadows ? 'soft' : false}
        camera={{ position: [0, 1.5, 4], fov: 35 }}
      >
        {/* Scene contents added in Phase 2 */}
        <color attach="background" args={['#05060A']} />
      </Canvas>
    </div>
  );
}
```

`frameloop="never"` for reduced-motion users means R3F renders once (initial mount) and never again. `"demand"` for normal users means render only when `invalidate()` is called or props change.

## Task 3.4: Build `ScrollCamera.jsx` with section-anchored interpolation

**Files:**
- Create: `src/three/ScrollCamera.jsx`

- [ ] **Step 3.4.1: Write the camera rig**

```jsx
import { useRef, useEffect, useState } from 'react';
import { useFrame, useThree, invalidate } from '@react-three/fiber';
import * as THREE from 'three';
import { SECTION_IDS, SCENE_KEYFRAMES } from './config/sceneKeyframes';
import { useLenisProgress } from '../lib/lenisContext';
import { TIER_CONFIG } from './hooks/usePerfTier';

const smoothstep = (t) => t * t * (3 - 2 * t);

const tmpA = new THREE.Vector3();
const tmpB = new THREE.Vector3();
const tmpTargetA = new THREE.Vector3();
const tmpTargetB = new THREE.Vector3();

export default function ScrollCamera({ tier, onLampIntensityChange }) {
  const { camera } = useThree();
  const config = TIER_CONFIG[tier];
  const [sectionRects, setSectionRects] = useState(null);
  const progress = useLenisProgress();
  const startTimeRef = useRef(performance.now());

  // Measure section DOM positions once mounted + on resize.
  useEffect(() => {
    function measure() {
      const totalScroll = document.documentElement.scrollHeight - window.innerHeight;
      const rects = SECTION_IDS.map((id) => {
        const el = document.getElementById(id);
        if (!el) return null;
        const top = el.offsetTop;
        const height = el.offsetHeight;
        return {
          id,
          startProgress: top / totalScroll,
          endProgress: (top + height) / totalScroll,
        };
      });
      setSectionRects(rects);
    }
    measure();
    window.addEventListener('resize', measure);
    return () => window.removeEventListener('resize', measure);
  }, []);

  useFrame(() => {
    if (!sectionRects) return;

    // Find which two keyframes the current scroll progress sits between.
    let activeIdx = 0;
    for (let i = 0; i < sectionRects.length; i++) {
      if (sectionRects[i] && progress >= sectionRects[i].startProgress) {
        activeIdx = i;
      }
    }
    const fromKf = SCENE_KEYFRAMES[activeIdx];
    const toKf = SCENE_KEYFRAMES[Math.min(activeIdx + 1, SCENE_KEYFRAMES.length - 1)];
    const fromRect = sectionRects[activeIdx];

    let localT = 0;
    if (fromRect) {
      const span = fromRect.endProgress - fromRect.startProgress;
      localT = span > 0 ? (progress - fromRect.startProgress) / span : 0;
      localT = Math.max(0, Math.min(1, localT));
    }
    const eased = smoothstep(localT);

    tmpA.fromArray(fromKf.cameraPos);
    tmpB.fromArray(toKf.cameraPos);
    camera.position.lerpVectors(tmpA, tmpB, eased);

    tmpTargetA.fromArray(fromKf.cameraTarget);
    tmpTargetB.fromArray(toKf.cameraTarget);
    const targetPos = tmpTargetA.lerp(tmpTargetB, eased);
    camera.lookAt(targetPos);

    // Idle jitter (mid + high only).
    if (config.idleJitter) {
      const t = (performance.now() - startTimeRef.current) / 1000;
      camera.position.y += Math.sin(t * 0.3) * 0.02;
    }

    // Broadcast interpolated lamp intensity for the scene to pick up.
    const lamp = THREE.MathUtils.lerp(fromKf.lampIntensity, toKf.lampIntensity, eased);
    if (onLampIntensityChange) onLampIntensityChange(lamp);
  });

  // Force a re-render every time Lenis progress changes (since frameloop is "demand").
  useEffect(() => {
    invalidate();
  }, [progress]);

  return null;
}
```
